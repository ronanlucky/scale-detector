What it does

Everything you need to analyse a song

🎯

Key detection

Identifies the musical key using the Krumhansl-Schmuckler algorithm with Pearson correlation across all 24 major and minor keys.

📊

Chromagram

Visualises pitch energy across all 12 note classes — root note highlighted in pink, scale notes in purple, out-of-scale notes in gray.

🎹

Transpose & pitch shift

Shift the detected key up or down by semitones. With the local server, actually pitch-shifts the audio in real time using librosa.

⚡

YouTube support

Paste any YouTube URL and the companion Python server downloads and analyses it automatically using yt-dlp and ffmpeg.

How it works

Three steps to find the key

1

Upload or paste URL

Drop an MP3, WAV, M4A or paste a YouTube link to download automatically.

2

Goertzel analysis

The audio is sliced into frames. Each frame is analysed for pitch energy across MIDI notes 36–95 using the Goertzel algorithm.

3

KS key detection

The chromagram is correlated against Krumhansl-Schmuckler major and minor profiles to find the best matching key.

Setup instructions

1
Install Python 3.10+
Download from python.org if you don't have it.
2
Put both files in the same folder
Place scale-detector.html and server.py in the same folder.
3
Start the server
python server.py
It auto-installs yt-dlp, ffmpeg and librosa on first run.
4
Open scale-detector.html in Chrome
Paste any YouTube URL, click Download & Analyze. Transpose buttons will pitch-shift the audio.
